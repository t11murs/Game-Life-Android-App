# Игра «Жизнь» (Conway’s Game of Life)

Учебный проект — реализация клеточного автомата **«Игра Жизнь»** Джона Конвея.  
Проект демонстрирует работу с алгоритмами, состояниями, тестированием и сборкой через Gradle.

---

## Описание проекта

«Игра Жизнь» — это двумерное поле клеток, каждая из которых может находиться в одном из двух состояний: **живая** или **мёртвая**.  
Состояние поля на следующем шаге вычисляется по фиксированным правилам, зависящим от количества соседей.

Проект реализует:
- хранение и обновление состояния игрового поля;
- пошаговую эволюцию поколений;
- тестирование корректности логики.

---

## Правила игры

Для каждой клетки на каждом шаге выполняются следующие правила:

- живая клетка с **меньше чем 2 живыми соседями** умирает;
- живая клетка с **2 или 3 живыми соседями** остаётся жить;
- живая клетка с **более чем 3 соседями** умирает;
- мёртвая клетка с **ровно 3 живыми соседями** становится живой.
